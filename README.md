# My math blog

A work-in-progress!

## Building the website

The following is based off the [Jekyll documentation](https://jekyllrb.com/docs).
On a fresh installation, install Ruby and excute
```
$ bundle install
```
and then prefix `bundle exec` with Jekyll commands to use the version defined in the Gemfile.

To update gems to their latest versions, run
```
$ bundle update
```

## Useful commands

Run a local server and build the site on every change:
```
$ bundle exec jekyll serve
```

- use the `--livereload` option to automatically refresh the browser window on change
- use the `--drafts` option to process and render draft posts

For more options, please see the [Jekyll build command options](https://jekyllrb.com/docs/configuration/options/#build-command-options).
